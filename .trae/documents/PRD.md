# 3D自动化立体车库模拟系统 - 产品需求文档 (PRD)

## 1. 项目概述

### 1.1 项目背景
本项目是一个基于Three.js的3D交互式自动化立体车库模拟系统，用于展示立体车库的存取车过程的可视化演示。系统通过3D动画展示车辆的横移、升降、平移等自动化操作，让用户直观了解立体车库的工作原理。

### 1.2 项目目标
- 实现3层5车位的立体车库3D可视化
- 模拟自动化存取车的完整流程动画
- 提供直观的车位状态管理
- 记录存取车历史数据
- 支持故障模拟演示

### 1.3 目标用户
- 停车场管理人员
- 立体车库设备采购方
- 技术演示和教育展示

---

## 2. 功能需求

### 2.1 3D场景展示
- **车库结构**：3层立体车库，每层5个车位，共15个车位
- **出入口**：车库前方设置车辆出入口
- **视角控制**：支持鼠标拖拽旋转视角、滚轮缩放
- **环境效果**：合适的光照和材质效果

### 2.2 自动取车功能
- **车牌号输入**：用户输入车牌号
- **车位定位**：系统自动查找车辆所在车位
- **取车动画**：
  - 升降平台升降
  - 横移机构横移
  - 平移机构将车辆运送到出入口
- **动画顺序**：横移 → 升降 → 平移

### 2.3 手动存车功能
- **空车位选择**：用户点击空车位（绿色）
- **存车动画**：车辆从出入口自动存入选中车位
- **状态更新**：车位状态从空闲变为占用

### 2.4 车位状态显示
- **空闲状态**：绿色显示
- **占用状态**：红色显示
- **实时更新**：存取车完成后自动更新状态

### 2.5 车牌号搜索
- **搜索功能**：输入车牌号搜索车辆
- **信息显示**：显示车位编号和存入时间
- **搜索结果**：高亮显示对应车位

### 2.6 存取记录报表
- **记录字段**：车牌号、操作类型（存/取）、时间、车位编号
- **报表展示**：表格形式展示所有记录
- **时间排序**：按时间倒序排列

### 2.7 故障模拟
- **故障按钮**：每个车位对应故障按钮
- **故障状态**：点击后车位停止响应
- **报警效果**：车位闪烁报警
- **故障解除**：再次点击解除故障

## 3. 用户界面设计

### 3.1 布局结构
- **左侧**：3D场景区域（占70%宽度）
- **右侧**：控制面板（占30%宽度）

### 3.2 控制面板模块
1. **操作区**
   - 车牌号输入框
   - 取车按钮
   - 搜索按钮

2. **车位状态区**
   - 15个车位状态列表
   - 每个车位显示：编号、状态、故障按钮

3. **搜索结果区**
   - 车辆信息展示
   - 车位编号
   - 存入时间

4. **存取记录区**
   - 记录表格
   - 分页或滚动展示

## 4. 交互设计

### 4.1 存车流程
```mermaid
flowchart LR
    A["用户点击空车位"] --> B["系统检查车位状态"]
    B --> C["启动存车动画"]
    C --> D["车辆从入口进入"]
    D --> E["升降+横移+平移"]
    E --> F["车辆到达指定车位"]
    F --> G["更新车位状态为占用"]
    G --> H["记录存车记录"]
```

### 4.2 取车流程
```mermaid
flowchart LR
    A["用户输入车牌号"] --> B["系统查找车辆位置"]
    B --> C["启动取车动画"]
    C --> D["横移+升降+平移"]
    D --> E["车辆到达出入口"]
    E --> F["更新车位状态为空闲"]
    F --> G["记录取车记录"]
```

### 4.3 故障模拟流程
```mermaid
flowchart LR
    A["用户点击故障按钮"] --> B["车位进入故障状态"]
    B --> C["车位闪烁报警"]
    C --> D["拒绝存取车操作"]
    D --> E["再次点击解除故障"]
```

## 5. 非功能需求

### 5.1 性能需求
- 3D场景帧率稳定在30fps以上
- 动画流畅无卡顿
- 响应时间小于100ms

### 5.2 兼容性
- 支持主流浏览器（Chrome、Firefox、Edge）
- 响应式布局适配不同屏幕

### 5.3 可用性
- 操作直观简单
- 动画过程清晰可见
- 状态反馈及时明确
